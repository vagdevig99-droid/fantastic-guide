# Agent‑First Banking Interface with Multi‑Agent Orchestration

## 1. Project Vision

This project demonstrates an **agent‑first banking experience**, inspired by interfaces such as *Revolut Air* — where the **AI agent is the primary interface**, not an add‑on feature or widget.

Instead of navigating menus, users interact with **one intelligent front‑door agent** that can:
- understand intent
- reason across banking contexts
- orchestrate multiple specialized agents
- safely execute or escalate actions

This is a **demo/reference implementation**, not a production system.
